# 🌙 Islamic Web Calendar & Ramadan Smart Alarm

Sebuah aplikasi web kalender Islami interaktif yang menggabungkan estetika premium, penunjuk waktu ganda (Masehi & Hijriah), serta sistem hitung mundur (*countdown*) otomatis menuju waktu shalat dan berbuka puasa. Aplikasi ini dirancang ramah pengguna selama 24 jam dengan fitur redup otomatis (*Smart Dimming*) saat malam hari tiba.

## ✨ Fitur Unggulan

* **⏱️ Real-time Countdown Timer:** Penghitung mundur dinamis setiap detik menuju waktu shalat terdekat (Sangat membantu untuk memantau waktu Imsak dan Berbuka Puasa).
* **📅 Dual Calendar System:** Menampilkan penanggalan Masehi yang sinkron dengan penanggalan Hijriah.
* **🌙 Smart Dimming (Day/Night Theme):** Menggunakan logika waktu lokal perangkat. Website akan otomatis meredup (*dark mode*) yang nyaman di mata setelah jam 18:00 sore hingga jam 05:00 pagi tanpa perlu tombol manual.
* **🎨 Premium Islamic Aesthetics:** Desain minimalis dengan aksen warna emas (`#D4AF37`), latar belakang nyaman, serta ornamen garis ganda khas geometris Islami.
* **🔊 Audio Bypass System:** Menyediakan tombol "Tes Suara" untuk mengaktifkan izin audio pada browser modern agar alarm tidak terblokir.
* **📱 Responsive Design:** Tampilan fleksibel yang rapi saat diakses melalui smartphone maupun laptop/desktop.
* **💬 Floating WhatsApp Partnership:** Tombol interaktif di pojok kanan bawah dengan efek animasi denur untuk mempermudah calon mitra menghubungi untuk kerja sama bisnis.

---

## 🛠️ Tech Stack (Teknologi yang Digunakan)

* **HTML5:** Untuk menyusun kerangka visual halaman web.
* **Tailwind CSS (CDN):** Untuk penataan desain modern yang cepat, rapi, dan responsif.
* **Vanilla JavaScript:** Untuk menggerakkan logika jam digital, kalender dinamis, perhitungan mundur, sistem audio, dan deteksi tema siang/malam.
* **FontAwesome:** Untuk integrasi ikon visual yang estetis.

## 🚀 Cara Menjalankan Proyek

1. **Clone** atau **Download** seluruh file di repositori ini.
2. Pastikan file `index.html`, `style.css`, dan `script.js` berada dalam **satu folder yang sama**.
3. Klik dua kali pada file `index.html` untuk membukanya di browser kesayangan Anda (Chrome, Edge, Safari, Firefox).
4. *Selesai! Aplikasi siap digunakan secara offline maupun online.*
